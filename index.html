Tehnologii 3D:

  • 3D Studio Max 3ds Max este un program de grafică pe computer pentru crearea de modele 3D, animații și imagini digitale. Este unul dintre cele mai populare programe din industria graficii pe computer și este bine cunoscut pentru că are un set de instrumente robust pentru artiștii 3D. Un favorit printre dezvoltatorii de jocuri, studiourile comerciale TV și arhitecții, 3ds Max este deținut de Autodesk, aceeași companie responsabilă pentru programe precum Maya și AutoCAD.
  • Blender Blender este un set de instrumente gratuit și open-source pentru grafică pe computer 3D, utilizat pentru crearea de filme animate, efecte vizuale, artă, modele imprimate 3D, grafică în mișcare, aplicații 3D interactive, realitate virtuală și jocuri pe calculator.
  • Maya Maya este o aplicație grafică 3D pe computer care este utilizată în dezvoltarea de jocuri video, aplicații 3D, filme animate, seriale TV și orice efecte vizuale. Acest software poate crea modele grele, care ne ajută să oferim efecte Ultra 3D care creează o vedere realistă la nivelul utilizatorului.
  • ZBrush ZBrush este folosit pentru crearea de modele „de înaltă rezoluție” (capabile să atingă peste 40 de milioane de poligoane) pentru a fi utilizate în filme, jocuri și animații, de către companii, de la ILM și Weta Digital, la Epic Games și Electronic Arts.
